# Stack and its operations

A stack is a linear data structure that uses LIFO (Last in first out) order. Stacks are
commonly used in memory managament, expression evaluation, recursion etc

Stacks mainly comprise of two basic operations **push** & **pop** but certain implementation may allow more advanced operations

- **push()** :add an item into the stack
- **pop()** :removes the top-most item from the stack
- **display()** :for logging the values inside the stack
- **size()** :to get the space occupied by the stack

```ts:stack.ts
interface IStack<T> {
  push(T: any): void;
  pop(): T | undefined | null;
  size(): number;
  display(): void;
}

```

I have used interfaces,generics and classes for this implementation

```ts:stack.ts
class Stack<T> implements IStack<T>{
  private storage: T[] = [];
  private capacity: number = 0;
  constructor(size: number) {
    this.capacity = size;
  }
  push(item: T) {
    if (this.storage.length === this.capacity) {
      throw new Error("Overflow")
    }
    this.storage.push(item)
  }
  pop() {
    return this.storage.pop()
  }
  size() {
    return this.storage.length
  }
  display() {
    console.log(this.storage)
  }
}

const stack = new Stack(3)
stack.push(1)
stack.push(11)
stack.push(124)
stack.display()
stack.pop()
stack.display()

```
